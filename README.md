# Sample_iTween_HeadingPath

## What is this?

iTween sample for unity5.    

## Feature

Object look ahead on moving iTween path.    
Use 'iTween Visual Editor' to make iTween path.    


## How to use

See blog: iTweenパス上を前を向きながら移動するサンプル
